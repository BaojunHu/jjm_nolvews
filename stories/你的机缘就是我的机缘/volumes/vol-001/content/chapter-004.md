# 第004章 护送委托

任务堂今天比平时更吵。

门口挂着新放出来的委托牌，木牌边缘还带木屑。林逸挤进去看了一眼，最上面那块写着“丁级护送，药商杜衡，青石驿来回”。酬劳不高，风险评语写着“路况复杂，偶有散修扰动”。

“偶有”这两个字，很像在说“看运气”。

林逸看完木牌，转头找韩执事。

韩执事正在案后盖章，抬眼见他，先把手里的章按完才开口：“你们接这单？”

“苏师姐说接。”

“她人呢？”

“在外面和药商谈车队配置。”

韩执事把登记簿推过来：“签名。提醒一句，这单昨天夜里被改过线路，内档才看得见。你要是会看图，自己多看两眼。”

林逸笔尖一顿：“改线理由？”

“写的是山体滑坡。”

“真滑了？”

“我没去看。”韩执事压低声音，“任务堂最近三次‘滑坡’，有两次查出人为。你心里有数。”

林逸点头，签完字把登记簿推回去：“谢谢执事。”

“别谢。活着回来再谢。”


+出门时，苏沐晴已经和药商杜衡把交接谈完。杜衡四十来岁，身材圆胖，手指上有常年拨算盘留下的茧，笑起来很客气，眼底却藏不住紧张。
+
+“苏仙子，林小哥。”杜衡拱手，“货都装好了，咱们辰末出发，日落前到北岭坡扎营，明日午前入青石驿。”
+
+林逸问：“原路线不是走东谷吗？”
+
+杜衡叹气：“东谷封了，说是塌方。小商队没资格申请复检，只能绕。”
+
+苏沐晴接过话：“绕线期间，你的人听我指挥。车队队形我来定。”
+
+杜衡连忙点头：“都听仙子的。”
+
+林逸站在旁边，看似发呆，实际在记车队配置：六辆货车，两辆空车，一辆备用水车，车夫九人，护卫四人，药徒两人。对丁级护送来说，人不少。
+
+人多不代表安全。
+
+人多代表消息更容易漏。
+
+
+车队出发后，苏沐晴骑在前侧，林逸步行跟着第二辆货车。杜衡每隔半个时辰就来确认“速度是否合适”，问法委婉，眼神总往两侧山林飘。
+
+林逸走了半天，鞋底全是尘，心里那根线越拉越紧。
+
+紧的不是敌意，是“被人看着”的感觉。
+
+这种感觉来自细节。比如路边石桩上的灰被新刮过，像有人前夜刚踩点；比如某些岔口有半截折枝，折口朝向一致，像故意留的路标；再比如每过一段就会出现一小团不该在这个季节冒头的灰色苔斑。
+
+苔斑是湿地才有。
+
+这条路本该偏干。
+
+林逸在中午休整时借口打水，绕去后方岔口看了看。回头时，他手里多了一张从石缝里抠出来的旧图角。
+
+图角很薄，边缘烂得厉害，正中有一枚细小三叉记号。
+
+他把图角折进袖口，没声张。
+
+
+午后继续赶路，天色到申时时压了层阴云。车队到一处岔口，前方是两条路：左线平缓但绕远，右线近却要穿一段狭谷。
+
+杜衡骑着矮马跑过来：“按任务堂给的图，走右线。”
+
+林逸看了眼右线入口，谷口风很怪，吹进不吹出。
+
+他又看地图，地图边角印着任务堂章，章没问题。问题在一处微小改笔：原本左线旁的“荐”字被刮掉，改成了右线。
+
+改笔很细，像熟手干的。
+
+林逸走到苏沐晴马旁，压低声音：“右线像笼子。”
+
+苏沐晴没回头：“依据。”
+
+“风向、改笔、路标。”
+
+“还有吗？”
+
+“有。太合理了。合理到像专门给我们选。”
+
+苏沐晴勒住缰绳，转头看杜衡：“改走左线。”
+
+杜衡脸一白：“可任务单……”
+
+“任务单是建议，不是天条。”
+
+“左线要多走两刻。”
+
+“那就多走。”
+
+杜衡犹豫了半息，还是点头：“听仙子的，改左线！”
+
+车夫们掉头时，林逸站在岔口后侧，盯着右线谷口看了一会。云层压低，谷里黑得快，像一口提前张开的锅。
+
+他没再看，转身跟上车队。
+
+
+改线后第一段路很平静，平静到让人发困。杜衡本来提着的心稍稍放下，还拿了两块肉干给林逸。
+
+“林小哥，刚才多亏你。”
+
+“先别谢太早。”
+
+“你还看出啥了？”
+
+“看出我们可能避开了第一口刀。”
+
+杜衡手一抖，肉干差点掉地上：“第一口？”
+
+“嗯。后面还有没有，要看对面耐心。”
+
+杜衡脸色一下又白回去：“我只是运药，不至于吧？”
+
+“你不至于。苏师姐至于。”
+
+杜衡沉默下来，不再问。
+
+
+傍晚前，车队抵达北岭坡。这里地势开阔，能看到远处河道，适合扎营。苏沐晴下令车排成半弧，火堆分四处，护卫两班轮值。她安排得很细，杜衡照办得很快。
+
+林逸趁众人忙碌，把那张图角拿出来对照任务图。
+
+三叉记号在图角边缘。
+
+任务图右下也有同款，只是被墨点盖了一层。
+
+同一个记号出现两次，不像巧合。
+
+他把图角递给苏沐晴。苏沐晴看完，眼神沉下去：“哪来的？”
+
+“午后岔口石缝。”
+
+“你没告诉我。”
+
+“当时还不够证据。”
+
+“现在够了？”
+
+“够你提高警戒。”
+
+苏沐晴把图角折进掌心，灵火一过，纸角化灰。她看着火星散掉，声音很低：“盯梢者不止一批。”
+
+“我也这么想。”
+
+“第一批改路线，第二批等我们改线后再判位置。”
+
+“第三批可能只记录，不出手。”
+
+苏沐晴抬眼看他：“你把他们分得很细。”
+
+“分细了，才知道该打谁、该放谁。”
+
+
+入夜后，营地逐渐安静。车夫们围着火堆吃干粮，护卫在外围来回走。杜衡把账册抱在怀里，坐哪都像怕被人偷走。
+
+林逸靠在车轮边，看似发呆，耳朵一直听着营地外的动静。风声里夹着几次细碎石响，距离远，像有人在试探步点。
+
+他起身走到苏沐晴旁边：“今晚会来人。”
+
+苏沐晴正在布最后一枚警符，头也没抬：“几个人？”
+
+“暂时听见两拨。真动手的可能三到四人。”
+
+“目的？”
+
+“未必是劫货，更像试刀。”
+
+“试谁？”
+
+“先试你，再看我。”
+
+苏沐晴把符按进地面，符线亮了一下又隐去：“你站后手位，不许逞能。”
+
+“我一直很稳。”
+
+“你稳？”
+
+“相对稳。”
+
+苏沐晴哼了一声，算默认。
+
+
+两人回到火堆旁时，杜衡凑过来，小心翼翼问：“苏仙子，今夜要不要加钱请临时护卫？”
+
+林逸先回：“现在加不到。你要真想花钱，明早请我们吃顿热面。”
+
+杜衡愣住：“就这？”
+
+“就这。人饿着时容易判断失误。”
+
+杜衡连忙点头：“行，行，到了驿站我包三顿。”
+
+苏沐晴看了林逸一眼，没说话。
+
+
+夜更深，云把月光挡得七七八八。营地外的黑像一层厚布，风一吹就鼓起来。
+
+林逸坐在火边，手里拿着那张任务图反复看，指尖停在被改过的右线“荐”字上。他把指腹压上去，能摸到细微凹凸。
+
+这不是临时涂改。
+
+是提前准备好的版面替换。
+
+换句话说，盯梢者在任务发出前就能动手。
+
+这条信息比“今晚有人来”更麻烦。
+
+他抬头看苏沐晴，苏沐晴也在看他。
+
+两人没有说破，同步把视线移向营地外侧。
+
+该来的快来了。
+
+
+子时前一刻，第一声石子滚落从坡下传来。
+
+护卫立刻握刀，杜衡抱着账册蹲到车后，动作熟练得让人心疼。火光被风吹得歪斜，几道影子在坡边拉长。
+
+苏沐晴拔剑半寸，剑锋映着火色，声音很清：“来都来了，别躲。”
+
+黑暗里有人笑了一声，没回应。
+
+林逸把任务图折好塞进袖里，手掌贴在车辕，心里那根线绷到最紧。
+
+岔路改线、同款标记、夜间试探，全串上了。
+
+他确定一件事。
+
+盯他们的人，不止一批。
+
+今夜这一拨，只是先头。
+
+下一拨，才是账目真正开张的地方。
