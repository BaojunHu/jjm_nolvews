# 第003章 随行第一单

寒潭外线这条路，林逸以前走过很多次。

那时他是外门杂役，背着药筐、拎着绳索，最多在潭边远远看一眼，等内门弟子取完东西再收尾。今天不一样，他跟在苏沐晴身后，腰间挂着新领的通行令，肩上没有药筐，只有一只空袋。

“空袋干什么用？”苏沐晴侧目。

“装意外之财。”

“你倒是乐观。”

“我叫提前准备。”

寒潭地势低，气温比别处冷一截。石阶尽头有块小平台，平时是弟子交接物资的地方。今天平台边围了几个人，一个外门弟子正抱着木匣，满脸通红，像刚挨过训。

见苏沐晴过来，几人连忙行礼。

“苏师姐。”

苏沐晴点头：“怎么回事？”

那外门弟子赶紧上前：“弟子赵成，奉药库令送‘寒骨草’到内门丹房。路上匣锁松了，草叶外露，怕坏药性，不敢乱动。”

林逸看了眼木匣。匣子不大，缝里透出一缕淡青光，确实是寒骨草。此草娇气，见热就蔫，见浊气就黑，运送时要全程用净寒符封着。赵成手抖得厉害，符纸贴歪了两张，难怪慌。

苏沐晴伸手：“拿来。”

赵成双手递匣。

就在这时，平台侧面忽然有人叫他名字：“赵成，你师父找你！”

赵成下意识回头，手上力道一散，木匣脱手。

匣盖弹开。

一株寒骨草连根带土飞出来，在空中划了道弧。

所有人都愣了半拍，随后齐刷刷去抢。

抢是本能。

谁都知道寒骨草值钱。

林逸没往前扑，他只抬了下手。那株草落下时，正好砸进他手里那只空袋。袋口一收，寒气被兜住，草根没伤。

空袋本来是普通布袋。

入草的一瞬，袋口契印一亮，布纹上多了一层淡淡符光。

苏沐晴离得最近，看得最清。她眼神一沉，先没说话，抬手压住围上来的人：“都退开。”

赵成脸都白了：“苏师姐，弟子不是故意的！”

“我知道。”

“那寒骨草……”

苏沐晴看向林逸：“拿出来。”

林逸解开袋口，把寒骨草捧出来。草叶完好，寒雾比刚才还稳。

围观几人面面相觑。

有人低声嘀咕：“怎么落他手里反而稳了？”

另一个回：“你问我，我问谁。”


+苏沐晴没有当场判归属，她先取出一只小型归属盘，盘边刻着内门任务堂印记，能做现场临测。她把寒骨草放在盘心，再把林逸和赵成的灵息各引一缕入盘。
+
+盘针转了两圈，停在林逸那侧。
+
+赵成傻眼：“不对啊！草是药库的，弟子只是送货！”
+
+林逸在旁边补一句：“我也只是接了一下。”
+
+这话比不说还扎心。
+
+苏沐晴抬手，示意大家闭嘴。她盯着盘针，声音平稳：“归属盘判的是‘当下结算口’，不是最终所有权。赵成，责任仍在你送达；林逸，临时持有。”
+
+赵成连连点头，明显松了口气。
+
+林逸也点头：“临时我擅长。”
+
+苏沐晴扫他一眼：“别得意太早。”
+
+
+按流程，突发归属争议要去任务堂做记录。于是临时巡查变成押运复核，队伍从寒潭折回半山。
+
+路上赵成一直跟在后面，小声重复“弟子失手、弟子该罚”。林逸听了半路，转头递过去一张净寒符：“先把匣子封好，再认罚。药坏了你罚得更多。”
+
+赵成愣住，赶紧接符：“多谢林师兄。”
+
+“我不是你师兄。”
+
+“那……林前辈？”
+
+林逸差点被这称呼噎住：“叫名字。”
+
+赵成老老实实点头：“林逸。”
+
+苏沐晴在前面听见，语气平平：“你人缘涨得挺快。”
+
+“我只是怕他把草再摔一次。”
+
+
+任务堂里今天人比昨天更多。争议窗口前排着七八人，韩执事远远看到苏沐晴，先招手让他们走内侧案台。
+
+“又异常？”韩执事问。
+
+苏沐晴把归属盘和寒骨草放下：“现场失手飞落，从契接住，盘判临时归属偏向从契。”
+
+韩执事嘴角抽了下：“你们这对契约，出门就给我加班。”
+
+林逸客气回应：“辛苦执事。要不我给你送点茶？”
+
+“免了。”韩执事翻开表格，“赵成，陈述经过。”
+
+赵成把前因后果说了一遍，越说越紧张，生怕被判故意失责。韩执事听完，按规记录，再做一次静态归属测算。结果和现场一致：当前结算口偏向林逸。
+
+韩执事叹气：“按规，寒骨草先由林逸暂管，送达丹房后再结最终归属。苏师侄，你担保流程。”
+
+“我担保。”
+
+“林逸，你签字。”
+
+林逸拿笔签名，顺手看了眼表头：临时归属争议单。
+
+他问：“这个单子会进公开档吗？”
+
+韩执事抬头：“你想公开？”
+
+“我不想。”
+
+“那就走内部档。”
+
+“多谢。”
+
+“别谢太早。类似情况出现三次，执法堂会强制介入。”
+
+“明白。”
+
+
+从任务堂出来，已近午后。
+
+苏沐晴带林逸和赵成去丹房交接。丹房管事验完寒骨草，确认药性无损，收草入库，给赵成开了免责半条，给林逸盖了“临管已移交”。
+
+赵成如释重负，临走前朝林逸深鞠一躬：“今天要不是你接住，我就完了。”
+
+林逸摆手：“别谢我，谢你手滑得刚好。”
+
+赵成没听懂，还是道了三次谢才跑。
+
+苏沐晴看着他背影，忽然问：“你为什么帮他？”
+
+“顺手。”
+
+“你这种人，很少做纯顺手的事。”
+
+林逸笑笑：“好吧。帮他有两个好处。第一，少一个会到处说我‘抢草’的敌人。第二，流程里多一个愿意作证的人。”
+
+“这才像你。”
+
+“你夸我？”
+
+“我在提醒你，算计别太吵。”
+
+
+回别院路上，苏沐晴把那张“临管已移交”单子递给他：“收好。以后这种单子都留副本。”
+
+林逸接过，折进账册夹层：“我正有此意。”
+
+“你今天接草时，契印亮得很快。”
+
+“我也感觉到了。”
+
+“像预判，不像反应。”
+
+“也许是我手快。”
+
+“你信？”
+
+林逸想了下：“一半信，一半不信。另一半留着观察。”
+
+苏沐晴没有继续追问，只说：“从今天起，涉及机缘交接，你站我右后侧半步。别乱跑。”
+
+“为什么右后？”
+
+“我顺手。”
+
+林逸点头：“合理。”
+
+
+晚饭后，林逸在外厅摊开今天所有单据：现场复核、任务堂争议单、丹房移交单。他把三张单按时间排好，写下注释。
+
+“事件：寒骨草脱手。”
+
+“触发：第三方干扰导致原持有人失去控制。”
+
+“落点：从契接触并实际承受风险，结算口偏移。”
+
+“处置：主方压争议，走流程闭环。”
+
+写完，他在页角加了一句：
+
+“苏沐晴已默认‘让林逸先接触机缘’的战术可行性。”
+
+这句是重点。
+
+昨天两人还是互相试探，今天已经出现了第一次默契。不是信任，是效率。苏沐晴发现把争议先压住再处理，损失最小；林逸发现只要流程闭环，异常就能变成可记录收益。
+
+卖身契开始像合作协议。
+
+
+夜深时，别院外传来几声窃语。林逸推窗看了一眼，两个路过弟子正低声聊“苏沐晴身边新随行能接歪机缘”。见窗开，两人立刻散了。
+
+林逸把窗关上，笑了下。
+
+消息已经传开。
+
+这不是坏事。
+
+坏事是传得太快太细，让不该注意的人也注意到。好事是传得刚够，用来挡住部分明枪，让对手误判他只是“运气古怪”。
+
+他回到案前，在账册页底写下今天结论：
+
+“第一单完成。收益不大，样本价值高。”
+
+“下一步：验证是否可主动设计‘低风险接触位’。”
+
+笔尖停住，他又补一行。
+
+“备注：苏沐晴从不耐烦转为观察，关系状态更新为‘可用变量’。”
+
+
+这时门外传来敲门声。
+
+“林逸，开门。”
+
+是苏沐晴。
+
+林逸把账册合上，起身开门。苏沐晴站在门外，手里拿着一枚木牌。
+
+“明日任务堂有护送委托，药商车队，路程两日。你跟我去。”
+
+林逸接过木牌，牌面写着“丁级护送”。
+
+“看着普通。”他说。
+
+“我也这么看。”苏沐晴淡淡道，“所以更要去。”
+
+“你怀疑有问题？”
+
+“从你接到寒骨草那刻开始，问题就不会少。”
+
+林逸点头：“明早几时？”
+
+“辰初，在任务堂集合。”
+
+“收到。”
+
+苏沐晴转身要走，走到廊下又回头补了一句：“还有，明天出门别带空袋。”
+
+林逸愣了下：“为什么？”
+
+“你每次带空袋，都会装到东西。”
+
+“这也怪我？”
+
+“怪你手太快。”
+
+她说完就走。
+
+林逸关上门，低头看手里的护送木牌，指腹在“丁级”两个字上摩了摩。
+
+丁级任务，最适合藏针。
+
+他把木牌放进袖中，吹灭灯前又看了一眼账册封面。
+
+今天这单不大。
+
+可它证明了一件事。
+
+契约不是只会把他锁死，也会在关键时刻把门推开一条缝。
+
+缝不大，够进手。
+
+够他先拿第一把钥匙。
